La boucle for Une boucle permet de performer une action tant que son cycle d'itérations n'est pas rempli par sa condition. Une boucle for contient trois « phases » à l’intérieur du couple de parenthèses : une phase d’initialisation, une phase de test (condition de sortie) et une phase d’itération (généralement une incrémentation). Chaque phase est séparée des autres par un point-virgule.Syntaxe:
function cycleSolaire(param){ let iterations= param; for(interations = 0; iterations <= 365; iterations++){ console.log("Une nouvelle journée s'achève"); if(iterations===365){ console.log("Une révolution de la terre autour du soleil vient d'être accomplie") }
